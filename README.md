# Skin Microbiome

A React-based dashboard to explore, filter, and sort skin microorganism data.

## Tech Stack
- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

## Local Development

1. Clone the repository:
   ```bash
   git clone [https://github.com/oleksandrhaiu/skin-microbiome.git](https://github.com/oleksandrhaiu/skin-microbiome.git)

2. Install dependencies:
   ```bash
   npm install

3. Run the development server:
   ```bash
   npm run dev

4. Access the app at http://localhost:3000.
