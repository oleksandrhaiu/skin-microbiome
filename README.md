# Skin Microbiome Explorer

A specialized React-based dashboard designed to explore, filter, and sort skin microorganism data. This project demonstrates core front-end development skills: state management, data transformation, and responsive UI design.

## Links
- **Live Demo**: [https://skin-microbiome.vercel.app/](https://skin-microbiome.vercel.app/)

## Key Features
- **Dynamic Filtering**: Easily switch between bacteria, fungi, or view all microorganisms.
- **Advanced Sorting**: Sort data by relative abundance in both ascending and descending order.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
- **Type Safety**: Built with strict TypeScript interfaces for strong data handling.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Deployment**: Vercel

## Local Development

1. **Clone the repository**:
    ```bash  
    git clone https://github.com/oleksandrhaiu/skin-microbiome.git
2. **Install dependencies:**
   ```bash  
    npm install
3. **Run the development server:**
   ```bash
   npm run dev
4. Access the app: Open http://localhost:3000 in your browser.
